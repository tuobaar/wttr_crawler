A {{elements[1]}} PERCENT DISPLAY OF THE ENTIRE WEATHER ARCHIVE

Instructions

  1. Note that the database records are listed as a percentage of the total database records. The page by default displays 5% of the latest entries.
  2. To view a portion or the entire database starting from the latest entries, enter a positive value in the input box and send or press enter. Example 'data_limit=5'
  3. To view a portion or the entire database starting from the oldest entries, enter a negative value in the input box and send or press enter. Example 'data_limit=-5'
  4. Enter any integer from -100 to 100. Integers less/greater than -100/100 would be set to -100/100 respectively. Floats/decimals/fractions, letters/alphabets/symbols are not allowed.
  5. Click here or on any active area in the header to return to the homepage or main menu.
