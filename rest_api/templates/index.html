Accessing Weather Archive:

A REST API for retrieving aggregated weather information from database.

Data crawled from 'https://wttr.in/'

API ENDPOINTS LINKS

Instructions

  1. Click on a desired endpoint link to display the respective aggregated data.
  2. A dashboard is provided at each endpoint. Change the value of the parameter(s) below:
    • date=2020-04-01
    • dates=2022-04-01,2022-04-02,2022-04-03
    • start_date=2022-04-01
    • end_date=2022-04-02
    • data_limit=5
  3. You can also copy and paste the links into your favourite browser or API platform such as Postman Client.

Access aggregated data for a single date.

http://127.0.0.1:5000/api/v1/resources/single_date_temp_wind?date=2022-04-01

Access aggregated data for multiple dates.

http://127.0.0.1:5000/api/v1/resources/multi_dates_temp_wind?dates=2022-04-01,2022-04-02,2022-04-03

Access aggregated data for interval dates.

http://127.0.0.1:5000/api/v1/resources/interval_dates_temp_wind?start_date=2022-04-01&end_date=2022-04-02

View a portion/percentage of the entire database records.

http://127.0.0.1:5000/api/v1/resources/view_database?data_limit=5
