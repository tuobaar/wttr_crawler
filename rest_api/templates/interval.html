ENDPOINT FOR QUERYING INTERVAL DATES

Instructions

  1. To obtain aggregated data between 2 dates(an interval), enter the start and end dates in the input box and send or press enter. Example 'start_date=2022-04-01' and 'end_date=2022-04-02'
  2. You can also enter the same date for both parameters to obtain the aggregated data for that date.
  3. Please adhere to the date format.
  4. Click here or on any active area in the header to return to the homepage or main menu.
